# Keysight_N7752A
Script for Keysight N7752A Power Meter

Software / Drivers:
- Python 3.9+
- NI-VISA (National Instruments) or PyVISA + pyvisa-py
- Keysight IO Libraries Suite (for IO Monitor and USB/GPIB conection)

Python libraries:
- pyvisa>=1.13
- pyvisa-py>=1.11
- numpy>=1.24
- matplotlib>=3.7
